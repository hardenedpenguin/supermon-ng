# Archived LAMP/Manual Install Files

This directory contains files and documentation that were previously used for traditional LAMP (Linux/Apache/MySQL/PHP) or manual installations of Supermon-ng.

## Why are these files here?

Supermon-ng now supports **Docker-only** deployment for simplicity, maintainability, and modern best practices. All legacy scripts, installer files, and documentation related to manual or LAMP-based setup have been moved here to avoid confusion and to keep the main project tree focused on containerized deployment.

- If you need to reference or restore LAMP/manual support, you can find the old files here.
- These files are **no longer maintained** and may be removed in a future release.

## What to use instead

- Use the provided `Dockerfile` and `docker-compose.yml` for all new installations.
- See the main `README.md` for up-to-date Docker deployment instructions.
